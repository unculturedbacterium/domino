# Code of Conduct

Domino contributors and maintainers are expected to communicate respectfully,
welcome constructive technical disagreement, and focus criticism on the work
rather than the person. Harassment, discrimination, threats, publication of
private information, and deliberately disruptive behavior are not acceptable.

Project maintainers may edit or remove contributions that violate these
expectations and may temporarily or permanently restrict participation. Report
conduct concerns privately to the repository owner through the contact method
listed on the GitHub profile. Reports will be reviewed as promptly and
confidentially as practical.

These expectations apply in the repository, issue tracker, code review, and
other spaces where a participant is representing the Domino project.
